C언어

C언어는 운영체제나 시스템 프로그래밍에 널리 사용되는 강력하고 효율적인 프로그래밍 언어입니다. 하드웨어를 직접 제어할 수 있는 메모리 관리 기능과 빠른 실행 속도가 가장 큰 특징이며, 이후 등장한 많은 프로그래밍 언어들의 뼈대가 되었습니다.

파이썬 (Python)

파이썬은 문법이 간결하고 읽기 쉬워 초보자가 학습하기에 매우 적합한 고급 프로그래밍 언어입니다. 풍부한 라이브러리를 바탕으로 데이터 분석, 인공지능(AI), 웹 서버 개발, 자동화 스크립트 등 현대 IT 산업의 거의 모든 분야에서 폭넓게 활용되고 있습니다.

HTML / CSS

웹 개발의 가장 기본이 되는 핵심 기술입니다. HTML은 문서의 구조와 내용을 시맨틱하게 설계하고, CSS는 레이아웃, 색상, 애니메이션 등 웹 페이지의 시각적인 스타일을 아름답게 꾸며줍니다. 사용자와 상호작용하는 화면(Front-end)을 만들기 위해 필수적으로 알아야 합니다.
